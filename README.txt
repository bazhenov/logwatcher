Сборка
------

Для успешной сборки проекта вам нужен настроенный maven2. Maven должен быть настроен на использование локальных репозитоиев
FarPost'а, так как он исполдьзует зависимости из этого репозитория.

Так же для того чтобы прошли все тесты необходимо скопировать файл ./logwatcher-core/mysql.properties.example в ./logwatcher-core/mysql.properties
и заполнить в нем реквизиты подключения к тестовой базе данных.
